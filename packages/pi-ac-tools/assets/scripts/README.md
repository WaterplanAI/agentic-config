# script assets

This directory contains bundled helper and hook scripts used by `@agentic-config/pi-ac-tools`.

Current shipped surface:
- helper scripts for wrapper-backed skills
- `hooks/dry-run-guard.py` for dry-run runtime parity through `@agentic-config/pi-compat`
- `hooks/gsuite-public-asset-guard.py` for blocking accidental public GSuite sharing overrides through `@agentic-config/pi-compat`
